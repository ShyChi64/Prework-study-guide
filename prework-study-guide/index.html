HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Opening tags are represented by angle brackets associated with tag names, such as head.
  • Closing tags begin with a forward slash. Insert content between the opening tags and closing tags, so thw browser understands how to treat the content.
  • Element refers to the entire element from opening tag to closing tag.
  • Tag refers only to what appears inside the angle brackets.
